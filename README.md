### 🚨 Disclaimer & False Positives
Because AspisVault is a local security utility that handles sensitive data and encryption entirely offline without central registry code-signing certificates yet, some aggressive antivirus software or Windows SmartScreen might flag the installer as an "unknown publisher". 

This is a common **false positive** for independent offline software. You can safely bypass it by clicking *"More Info" -> "Run Anyway"*. Your data never leaves your machine.
